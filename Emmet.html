The use of Emmet !

Emmet is a plugin for many popular text editors which greatly improves HTML & CSS workflow. It is available for Sublime Text, Atom, Brackets, Visual Studio Code, WebStorm, PhpStorm, NetBeans, Eclipse, Dreamweaver, IntelliJ IDEA, and VIM. It is also available as a web application and as a standalone application for Windows, Mac OS X and Linux. Programming languages supported by Emmet include HTML, XML, XSL, and CSS, handlebars and PHP. Emmet also supports the creation of custom snippets for other languages.

Emmet Shortcuts (HTML)

  1. ! - Generates a new HTML5 doctype.

  2. Child ">" - Generates a child element nested between parent tags. eg. nav>ul>li

  3. Sibling "+" - Generates a sibling element nested between parent tags. eg. nav+ul+li

  4. Climb-up "^" - Generates a climb-up element nested between parent tags. eg. nav>ul>li^p

  5. Multiplication "*" - Generates a multiplication element nested between parent tags. eg. ul>li*5

  6. Grouping "()" - Generates a grouping element nested between parent tags. eg. (ul>li)*5

  7. Item Numbering "$" - Generates a item numbering element nested between parent tags. eg. ul>li.item$*5

  8. ID and Class "#" and "." - Generates a ID and Class element nested between parent tags. eg. ul#list>li.item$*5

  9. Text "[]" - Generates a text element nested between parent tags. eg. ul#list>li.item$*5[Item $]

  10. Custom Attribute "{}" - Generates a custom attribute element nested between parent tags. eg. a{Click me}

  11. Custom Attribute with Text "{text}" - Generates a custom attribute with text element nested between parent tags. eg. a{Click me}

  12. meta:utf - Generates a meta element with utf-8 charset. eg. meta:utf

  13. link:css - Generates a link element with css stylesheet. eg. link:css

  14. link:css+js - Generates a link element with css and js stylesheet. eg. link:css+js

  15. link:css+js+rel - Generates a link element with css, js and rel stylesheet. eg. link:css+js+rel
